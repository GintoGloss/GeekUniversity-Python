Создать файл file1 и наполнить его произвольным содержимым. Скопировать его в file2. Создать символическую ссылку file3 на file1. Создать жесткую ссылку file4 на file1. 
Посмотреть, какие айноды у файлов. Удалить file1. Что стало с остальными созданными файлами? Попробовать вывести их на экран.

ginto@ginto-VirtualBox:~$ cat > file1
random
content
12345
^C
ginto@ginto-VirtualBox:~$ cat file1 > file2
ginto@ginto-VirtualBox:~$ ln -s file1 file3
ginto@ginto-VirtualBox:~$ ln file1 file4
ginto@ginto-VirtualBox:~$ ls -ila file*
238925 -rw-r--r-- 2 ginto ginto 21 мар  6 18:00 file1
192575 -rw-rw-r-- 1 ginto ginto 21 мар  6 18:03 file2
166346 lrwxrwxrwx 1 ginto ginto  5 мар  6 18:10 file3 -> file1
238925 -rw-r--r-- 2 ginto ginto 21 мар  6 18:00 file4
ginto@ginto-VirtualBox:~$ rm file1
ginto@ginto-VirtualBox:~$ cat file2
random

ginto@ginto-VirtualBox:~$ cat file3
cat: file3: No such file or directory
ginto@ginto-VirtualBox:~$ cat file4
random

Дать созданным файлам другие, произвольные имена. Создать новую символическую ссылку. Переместить ссылки в другую директорию.

ginto@ginto-VirtualBox:~$ mv file2 somefile
ginto@ginto-VirtualBox:~$ mv file4 randomfile
ln -s randomfile  new_link
ginto@ginto-VirtualBox:~$ mv file3 Desktop/
ginto@ginto-VirtualBox:~$ mv new_link Desktop/
ginto@ginto-VirtualBox:~$ ls Desktop/
file3  new_link
ginto@ginto-VirtualBox:~$ cat Desktop/new_link
cat: Desktop/new_link: No such file or directory
ginto@ginto-VirtualBox:~$

Создать два произвольных файла. Первому присвоить права на чтение, запись для владельца и группы, только на чтение — для всех.
Второму присвоить права на чтение, запись — только для владельца. Сделать это в численном и символьном виде.

ginto@ginto-VirtualBox:~$ chmod ug=rw,o=r somefile
ginto@ginto-VirtualBox:~$ chmod 600 randomfile
ginto@ginto-VirtualBox:~$ ls -l randomfile
-rw------- 1 ginto ginto 21 мар  6 18:00 randomfile
ginto@ginto-VirtualBox:~$
ginto@ginto-VirtualBox:~$ useradd -G sudo myuser
